## Project Description

This project is a simple implementation of a bigram language model using Python.
The model predicts the next word based on the previous word.

To improve the model, smoothing techniques like Laplace and Kneser-Ney are used.
The performance of the model is checked using perplexity.

---

## Tech stack

* Python
* NLTK
* NumPy

---

## Dataset

A small text dataset is used from `data.txt`.
The text is converted into lowercase and then tokenized.

---

## How to Run

1. Install required libraries:
   pip install nltk numpy

2. Run the file:
   python main.py
